# Security Policy

## Supported Versions

We actively monitor and provide security patches for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do **NOT** open a public issue on GitHub. Instead, report it privately to ensure the safety of all users.

To report a vulnerability:
1. Send an email to the maintainer at: **[dvy.zhang@gmail.com](mailto:dvy.zhang@gmail.com)** or contact **[AliSakkaf](https://github.com/alisakkaf)**.
2. Include details of the vulnerability, including step-by-step instructions or proof-of-concept code to reproduce the issue.
3. We will acknowledge your report within 48 hours and work with you to patch the issue before disclosing it publicly.
